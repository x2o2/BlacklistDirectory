| [诈骗的](黑名单/诈骗的)             | 拉黑这些，能适当减少上当受骗                                    |
| [地区](黑名单/地区)               | 特定地区的广告甚多且有炒作特点所以拉黑了                              |
| [名](黑名单/名)                 |                                                   |
| [热点事件](黑名单/热点事件)           | 那些被炒作的热点希望让你看到的，这些并非我所愿 所以拉黑了                     |
